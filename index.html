Live Scoring Setup

🚀 GitHub Actions — Free, No API Key This site uses GitHub Actions to automatically fetch scores every 5 minutes and save them to a scores.json file in your repo. No serverless functions, no paid APIs.

One-time setup (2 minutes):
1. Your repo is already on GitHub Pages
2. Before each tournament, update tournament-id.txt in your repo with the ESPN tournament ID
3. The workflow runs automatically Thu–Sun during tournament hours

To find the ESPN tournament ID: Go to espn.com/golf/leaderboard, navigate to your tournament — the number at the end of the URL is the ID.
Example: espn.com/golf/leaderboard/_/tournamentId/401811938
Paste the number from the ESPN URL. This gets saved to your repo as tournament-id.txt.
